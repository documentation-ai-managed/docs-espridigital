## Overview

Espridigital empowers you to centralize, organize, and collaborate on your project documentation effortlessly. As a SaaS platform, it provides dedicated spaces for teams to create, version, and share docs without the hassle of scattered files or complex tools. You benefit from intuitive organization, real-time collaboration, and powerful search features that keep your knowledge accessible.

<Callout kind="info">
Espridigital integrates seamlessly with your workflow, supporting Markdown, rich text, and API-driven content updates.
</Callout>

## Quick Start

Get up and running with Espridigital in minutes. Follow these steps to create your first documentation space.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.espridigital.com` and create your account using email or SSO.

    <Callout kind="tip">
      New users receive a free trial with unlimited spaces.
    </Callout>
  </Step>
  <Step title="Create a Space" icon="plus">
    Click "New Space" in the dashboard. Name it (e.g., "Project Alpha Docs") and set visibility.
  </Step>
  <Step title="Add Your First Doc" icon="file-text">
    Upload Markdown files or start a new page. Use the editor to format content.

````javascript
// Example: Embed Espridigital docs in your app
const embedDoc = (spaceId, docId) => {
  const iframe = document.createElement('iframe');
  iframe.src = `https://docs.espridigital.com/embed/${spaceId}/${docId}`;
  iframe.width = '100%';
  iframe.height = '600';
  document.body.appendChild(iframe);
};
````

  </Step>
  <Step title="Share and Collaborate" icon="share-2">
    Generate share links or invite collaborators via email.
  </Step>
</Steps>

## Brand Guidelines

Follow these guidelines to maintain consistency when customizing Espridigital themes or embeds.

<Expandable title="Color Scheme" default-open="true">
  Use the primary brand color `#3B82F6` for accents and buttons.

  ```css
  :root {
    --espridigital-primary: #3B82F6;
    --espridigital-secondary: #1E40AF;
    --espridigital-bg: #F8FAFC;
  }
  ```

  Secondary colors include `#1E40AF` for depth and `#F8FAFC` for backgrounds.
</Expandable>

<Expandable title="Typography and Icons">
  Pair with sans-serif fonts like Inter or system defaults. Use Lucide icons for UI elements matching Espridigital's style.
</Expandable>
