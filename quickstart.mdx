## Prerequisites

Before you begin, ensure you have:

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Internet connection

<Callout kind="info">

Espridigital is free to start. No credit card required for the basic plan.

</Callout>

## Sign Up and Log In

Follow these steps to create your Espridigital account.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Open your browser and navigate to `https://dashboard.example.com`.
  </Step>
  <Step title="Register" icon="user-plus">
    Click **Sign Up**. Enter your email, create a strong password, and complete the CAPTCHA.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification email from Espridigital. Click the link to activate your account.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the dashboard, enter your credentials, and click **Log In**.
  </Step>
</Steps>

## Create Your First Documentation Space

A documentation space organizes your project docs. Create one now.

<Steps>
  <Step title="Access Spaces" icon="book-open">
    In the sidebar, click **Spaces** or the `+` icon.
  </Step>
  <Step title="New Space" icon="plus">
    Select **New Space**. Name it `My Project Docs` and choose a slug like `my-project`.
  </Step>
  <Step title="Configure" icon="settings">
    Set visibility to **Public** or **Private**. Add a description: "Documentation for my app."
  </Step>
  <Step title="Save" icon="check-circle">
    Click **Create Space**. You'll land in your new space dashboard.
  </Step>
</Steps>

## Basic Interface Navigation

Explore the main areas using these tabs.

<Tabs>
  <Tab title="Sidebar" icon="menu">
    - **Spaces**: Manage documentation spaces
    - **Documents**: List and search docs
    - **Settings`: Space configuration
  </Tab>
  <Tab title="Editor Toolbar" icon="edit">
    - Bold, italic, headings
    - Insert images, code blocks
    - Preview toggle
  </Tab>
  <Tab title="Preview Pane" icon="eye">
    Live rendering of MDX
    Responsive design check
    Shareable preview link
  </Tab>
</Tabs>
