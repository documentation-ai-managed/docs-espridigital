## Upgrading to the Latest Version

Follow these steps to update Espridigital smoothly and avoid disruptions.

<Steps>
  <Step title="Backup Your Projects" icon="database">

    Export all projects via the dashboard or API.

    ```bash
    curl -H "Authorization: Bearer YOUR_TOKEN" \
         https://api.example.com/v2/projects/export \
         -o backup.json
    ```

  </Step>
  <Step title="Update Client SDK" icon="download">

    <CodeGroup tabs="npm,yarn,pip">

    ````bash
    npm install espridigital@1.3.0
    ````

    ````bash
    yarn add espridigital@1.3.0
    ````

    ````bash
    pip install espridigital==1.3.0
    ````

    </CodeGroup>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Execute database migrations if self-hosting.

    ```bash
    docker exec espridigital migrate up
    ```

  </Step>
  <Step title="Verify and Test" icon="check-circle">

    Test key workflows like editing and sharing.

  </Step>
</Steps>

<Callout kind="tip">

  Always test upgrades in a staging environment first. Check the [migration guide](/quickstart#upgrading) for v1.1.0 breaking changes.

</Callout>
